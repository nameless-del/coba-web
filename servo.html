Proyek Servo

Deskripsi Proyek

Proyek ini bertujuan untuk mengendalikan pergerakan motor servo menggunakan dua tombol sederhana. Servo akan bergerak ke posisi 90 derajat ketika tombol pertama ditekan dan akan kembali ke posisi 0 derajat saat tombol kedua ditekan. Proyek ini cocok untuk pembelajaran dasar mikrokontroler dan aktuator sederhana, serta dapat dikembangkan menjadi sistem kontrol yang lebih kompleks.

Komponen yang Dibutuhkan

  • 1 Arduino (Uno, Mega, dsb.)
  • Servo motor
  • Dua tombol
  • Kabel jumper

Kode Arduino


#include <.Servo.h>  // Memasukkan pustaka Servo

Servo servoMotor;   // Membuat objek servo

const int pinButton90 = 7;    // Pin untuk tombol pertama (90 derajat)
const int pinButton0 = 8;     // Pin untuk tombol kedua (0 derajat)
const int posisiAwal = 0;     // Posisi awal servo (0 derajat)
const int posisiTujuan = 90;  // Posisi tujuan servo (90 derajat)

void setup() {
  servoMotor.attach(9);            // Menghubungkan servo ke pin digital 9
  pinMode(pinButton90, INPUT_PULLUP); // Mengatur tombol 90 derajat sebagai input dengan pull-up internal
  pinMode(pinButton0, INPUT_PULLUP);  // Mengatur tombol 0 derajat sebagai input dengan pull-up internal
  servoMotor.write(posisiAwal);     // Menetapkan posisi awal servo
}

void loop() {
  if (digitalRead(pinButton90) == LOW) {  // Jika tombol 90 derajat ditekan
    servoMotor.write(posisiTujuan);       // Menggerakkan servo ke posisi 90 derajat
  } 
  else if (digitalRead(pinButton0) == LOW) { // Jika tombol 0 derajat ditekan
    servoMotor.write(posisiAwal);         // Mengembalikan servo ke posisi 0 derajat
  }
}

Cara Kerja

  1. Tombol pertama (pin 7) menggerakkan servo ke posisi 90 derajat.
  2. Tombol kedua (pin 8) mengembalikan servo ke posisi 0 derajat.

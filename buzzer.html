Lagu Cicak Cicak Di Dinding

Deskripsi Proyek

Proyek ini menggunakan Arduino dan buzzer untuk memainkan lagu "Cicak-Cicak di Dinding". Proyek ini sederhana dan ideal untuk pemula yang ingin mempelajari pengendalian suara pada Arduino. Buzzer akan menghasilkan suara sesuai dengan nada lagu yang telah dikodekan dalam program.

Komponen yang Dibutuhkan

  • 1 Arduino (Uno, Mega, dsb.)
  • 1 buzzer
  • Kabel jumper

Gambar arduino buzzer

Kode Arduino


// Definisi frekuensi untuk setiap nada
#define NOTE_C4  262
#define NOTE_D4  294
#define NOTE_E4  330
#define NOTE_F4  349
#define NOTE_G4  392
#define NOTE_A4  440
#define NOTE_B4  494
#define NOTE_C5  523

// Melodi lagu "Cicak-cicak di Dinding" sesuai notasi dari gambar
int melody[] = {
  NOTE_G4, NOTE_E4, NOTE_G4, NOTE_E4, NOTE_E4, NOTE_F4, NOTE_G4, // Cicak-cicak di dinding
  NOTE_F4, NOTE_D4, NOTE_F4, NOTE_A4, NOTE_G4, NOTE_F4, NOTE_E4, // Diam-diam merayap
  NOTE_A4, NOTE_F4, NOTE_A4, NOTE_F4, NOTE_A4, NOTE_B4, NOTE_C5, // Datang seekor nyamuk
  NOTE_C5,                                                      // Hap
  NOTE_E4, NOTE_G4, NOTE_F4, NOTE_D4, NOTE_C4                   // Lalu ditangkap
};

// Durasi setiap nada (4 = seperempat, 8 = seperdelapan, dll.)
int noteDurations[] = {
  8, 8, 8, 8, 8, 8, 4,
  8, 8, 8, 8, 8, 8, 4,
  8, 8, 8, 8, 8, 8, 4,
  2,
  8, 8, 8, 8, 4
};

void setup() {
  // Memainkan setiap nada dalam melodi
  for (int thisNote = 0; thisNote < 27; thisNote++) {
    int noteDuration = 1000 / noteDurations[thisNote];
    tone(1, melody[thisNote], noteDuration);

    // Jeda antar nada
    int pauseBetweenNotes = noteDuration * 1.6;
    delay(pauseBetweenNotes);

    // Menghentikan nada
    noTone(1);
  }
}

void loop() {
  // Tidak perlu mengulang lagu
}

Cara Kerja

  1. Nada: Nada-nada dari lagu "Cicak-Cicak di Dinding" disimpan dalam array
  2. Durasi: Setiap nada memiliki durasi yang sama, disimpan dalam array
  3. Fungsi tone(): Digunakan untuk menghasilkan suara pada buzzer dengan frekuensi yang ditentukan.
  4. Perulangan: for loop digunakan untuk memainkan nada satu per satu sesuai durasi yang ditentukan.
